# FocusMode - Student Distraction-Free Learning

![FocusMode Logo](/icons/icon-192x192.png)

Aplikasi web progresif (PWA) yang dirancang untuk membantu siswa dan pelajar meningkatkan fokus belajar dengan teknik Pomodoro, pencatatan digital, dan manajemen materi belajar.

## 🚀 Fitur Utama

### 🎯 Focus Mode
- Timer Pomodoro (25/15/45 menit)
- Mode fokus penuh tanpa gangguan
- Notifikasi sesi selesai
- Statistik produktivitas

### 📝 Catatan Digital
- Buat dan kelola catatan belajar
- Sinkronisasi dengan REST API
- Penyimpanan offline
- Format teks lengkap

### 📚 Rak Buku Digital
- Kelola koleksi buku dan materi
- Kategori "Belum dibaca" dan "Selesai"
- Informasi detail buku
- Pindah antar rak dengan mudah

### 🎨 Sesi Belajar
- Rencanakan sesi belajar
- Lacak progres belajar
- Catatan untuk setiap sesi
- Statistik waktu belajar

### 💫 Progressive Web App
- Installable di desktop dan mobile
- Bekerja offline
- Push notifications
- Fast loading dengan service worker

## 🛠 Teknologi yang Digunakan

- **Frontend**: Vanilla JavaScript (ES6+), Web Components, CSS Grid & Flexbox
- **Storage**: IndexedDB, LocalStorage, REST API
- **PWA**: Service Worker, Web App Manifest
- **Build Tool**: Webpack 5
- **Development**: ESLint, Prettier
